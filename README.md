# PHP 深入浅出

PHP 是一种创建动态交互性站点的服务器端脚本语言。

PHP 是开源的，免费的。

后端环境：LAMP(Linux + Apache + MySQL + PHP) 或 LNMP(Linux + Nginx + MySQL + PHP)。
